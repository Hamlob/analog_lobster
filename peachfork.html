Peachfork is a module that provides a real-time pitch-shifting. This is applicable for any signal including polyphonic or non-harmonic signals such as drums or chords.

Control Type Function
BLEND POT Controls ratio of raw and pitch-shifted signal
SHIFT POT Selects the scaling of interval and input CV
LATCH BTN Changes mode between momentary ON and latched ON/OFF
OFFSET POT Adjusts the offset for 0V incoming EXP signal
SW BTN Manual control to turn ON/OFF the pitch-shifting (depends on LATCH)
IN IN Audio input
EXP IN CV input controlling the pitch-shift amount (can be both >0V or <0V)
EN IN Enable signal to turn ON/OFF the pitch-shifting (depends on LATCH)
OUT OUT Audio output
